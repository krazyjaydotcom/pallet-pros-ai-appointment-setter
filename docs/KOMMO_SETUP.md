# Kommo Setup Guide

## Actions the owner performs inside Kommo

1. Register the integration.
2. Select the required OAuth scopes.
3. Add the OAuth redirect URL.
4. Configure the Salesbot or private-chatbot step.
5. Select the incoming-message trigger.
6. Limit the trigger to Instagram.
7. Connect the application URL / widget request endpoint.
8. Test with a real Instagram DM.
9. Disable the automation when needed.
10. Switch operating modes from the dashboard when appropriate.

## Actions performed inside the application

1. Add credentials.
2. Connect Kommo.
3. Confirm account access.
4. Review incoming events.
5. Approve proposed responses.
6. Publish knowledge-base content.
7. Test the playground.
8. Enable limited automatic sending only after deliberate admin action.

## Placeholders to confirm in the live account

- Exact Kommo menu labels for the Salesbot and integration screens.
- Exact name of the private chatbot / widget setup path in the account.
- Exact trigger label for incoming Instagram messages.
- Exact callback URL field names and sample payload keys.
- Exact state for disabling all sending immediately.
